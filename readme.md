## Configurasi
- untuk menambahkan nomer owner, masuk ke file config.js
```json
{
 owner: ["272206605082689@lid", "179573169848377@lid"],
}
```
- untuk set id bot, masuk ke file config.js
 ```json
{
  admin: "179573169848377@lid",
}
```
- configurasi env, catatan bot ini menggunakan supabase sebagai databasenya
```env
NOXER=
DB_URL= 
DB_KEY= 
GROQ_API_KEY=
```
